\lessonheader{3}{15}{Patterns with Numbers 0–5}

\begin{instructbox}
\noindent\textbf{\color{sd-blue}Instruction} \textit{(read aloud):}\\[4pt]
Patterns repeat in a predictable way.
Numbers can make patterns too!
Today we find, continue, and create number patterns using 0 through 5.
Look carefully — what comes next?
\end{instructbox}
